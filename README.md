# Him Kriya - Professional Portfolio

A modern, responsive portfolio website for an IT Executive & Systems Administrator, designed for job applications and professional networking.

## Live Demo

The portfolio is deployed on GitHub Pages and can be accessed at:  
`https://[your-username].github.io/portfolio/`

## Features

- **Modern & Professional Design**: Clean layout with a professional color scheme
- **Fully Responsive**: Optimized for all devices (mobile, tablet, desktop)
- **Dark/Light Mode**: Toggle between dark and light themes
- **Interactive Elements**:
  - Smooth scrolling navigation
  - Hover animations
  - Fade-in section transitions
  - Mobile hamburger menu
  - Scroll-to-top button
- **Complete Professional Information**:
  - Hero section with call-to-action
  - About section with professional summary
  - Skills categorized by expertise
  - Work experience timeline
  - Projects showcase
  - Education and languages
  - Contact form and information

## Tech Stack

- **HTML5**: Semantic markup for accessibility
- **CSS3**: Modern CSS with Flexbox and Grid
- **JavaScript (Vanilla)**: No frameworks or libraries
- **Font Awesome**: For icons
- **Google Fonts**: Poppins and Roboto typography
